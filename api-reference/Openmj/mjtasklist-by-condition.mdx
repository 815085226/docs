## POST 根据ID列表查询任务

POST /mj/task/list-by-condition

> Body 请求参数

```json
{
  "ids": [
    "1761667386545025"
  ],
  "apiKey": "{{apiKey}}",
  "timestamp": "{{timestamp}}",
  "sign": "{{sign}}"
}
```

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|system|header|string| 是 |none|
|Content-Type|header|string| 是 |none|
|Authorization|header|string| 是 |none|
|body|body|object| 否 |none|
|» ids|body|[string]| 是 |none|
|» apiKey|body|string| 是 |none|
|» timestamp|body|string| 是 |none|
|» sign|body|string| 是 |none|
